# 💐 Digital Perfume Creator

> If feelings had a fragrance, this is what they would look like.

## ✨ Overview
Digital Perfume Creator is a creative web app where users design a “fragrance” using emotions instead of scents. Each perfume is made of three layers — Top, Heart, and Base notes — represented through colors, animations, and motion.

The final result is a shareable link where the recipient experiences a cinematic diffusion of visuals and messages.

## 🚀 Features
- Emotion-based perfume creation
- Layered note system (Top, Heart, Base)
- Smooth animated transitions
- Personalized message reveal
- Shareable unique links
- Responsive UI

## 🛠️ Tech Stack
- React
- Tailwind CSS
- Framer Motion
- Node.js (optional)
